# CNI-membership-map
A prototype mapping of CNI members to explore regions and other details

This map represents the locations and additional organizational information for CNI member organizations. More details on the organizations can be found at https://www.cni.org/about-cni/membership/members.

This map was developed by pairing with Claude AI to develop html and is presented for exploration. It can be further developed with your feedback and suggestions. Please share those thoughts - and report errors - as [pull requests](https://github.com/meghaninmotion/CNI-membership-map/pulls)!
